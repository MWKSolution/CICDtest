# pre-commit

1. pip install pre-commit
2. create file .pre-commit-config.yaml
3. pre-commit install - git hook scripts

```text
repos:
-   repo: ...
    rev: ...
    hooks:
    -   id: ...
    ...
-   repo: ...
    rev: ...
    hooks:
    -   id: ...
    ...
```
4. pre-commit run --all-files or just commit...
